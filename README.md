# YuDo
## What Is YuDo
YuDo Is A Simple ToDo List App.
#### Features
* Adding Tasks For Certain Dates
* Deletable Tasks
* Different Lists (Today, Tomorrow, Upcoming Next)
* Editable Lists
## What Will We Add
We Will Be Trying To Add A Way To Use Voice Commands And Widgets.
